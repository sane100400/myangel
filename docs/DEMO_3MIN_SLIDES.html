문제 정의

Problem

프롬프트는 애매하고, 부분 편집은 위치가 흔들립니다

  • "예쁜", "감성적인" 같은 표현은 결과를 예측하기 어렵습니다.
  • 자연어만으로 "그 의자"를 바꾸면 모델마다 참조 영역이 달라집니다.
  • 생성 결과를 저장·공유·비용 관리까지 이어주는 흐름이 필요합니다.
핵심 과제표현의 애매함과 편집 위치의 불확실성을 줄여야 합니다.
데모 목표생성부터 정밀 편집, 저장과 공유까지 3분 안에 한 번에 보여줍니다.

0:20-0:45 | 일반 생성 도구의 문제는 애매한 표현과 위치 불확실성입니다. 데모는 이 두 문제를 줄이는 흐름에 집중합니다.
