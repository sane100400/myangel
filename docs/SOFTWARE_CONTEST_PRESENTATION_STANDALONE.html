프롬프트부터 부분 편집까지 연결한 한국어 AI 이미지 스튜디오

MyAngel은 생성, 마커 기반 편집, 저장, 공유, 크레딧 운영을 하나의 제품 흐름으로 묶어 이미지 제작 시행착오를 줄입니다.

Problem

왜 필요한가

이미지 생성의 실패는 모델 성능보다 작업 흐름에서 많이 발생합니다

01 Prompt

한국어 프롬프트가 추상적이면 결과가 흔들림

"예쁘게", "감성적으로" 같은 표현은 모델이 매번 다르게 해석해 재생성 비용을 키웁니다.

02 Edit

부분 편집에서 위치와 의도가 분리됨

말로만 "오른쪽 의자 바꿔줘"라고 하면 모델은 어느 영역을 고칠지 안정적으로 알기 어렵습니다.

03 Ops

저장·공유·과금이 분리되면 서비스가 되기 어려움

데모가 실제 제품이 되려면 인증, 크레딧, 저장소, 공개 갤러리까지 한 번에 닫혀야 합니다.

Solution Flow

제품 해결 방식

프롬프트를 정리하고, 모델을 고르고, 마커로 편집하고, 결과를 운영합니다

사용자가 화면에서 하는 선택을 모델별 API 계약, 후처리, 저장, 크레딧 로직으로 안정적으로 변환합니다.

1. Generate한국어 프롬프트 강화와 레퍼런스 기반 생성
2. Model ChoiceGPT Image 2와 Nano Banana Pro를 한 화면에서 선택
3. Marker Edit교체·추가·삭제 영역을 원형 마커로 지정
4. Operate저장, Discover 공유, 크레딧 차감·환불 처리

Core Differentiator

핵심 기술

마커 편집 프로토콜로 "어디를 어떻게"를 함께 전달합니다

  • 1마커 좌표, 반지름, 작업 유형을 정규화된 데이터로 검증합니다.
  • 2Nano Banana Pro에는 원본, 마커 가이드, 그레이스케일 마스크를 함께 보냅니다.
  • 3GPT Image 2에는 원본과 알파 마스크를 보내고, 1K/2K 후처리만 적용합니다.
  • 4마커 없이도 전역 분위기·조명 조정은 별도 필드로 처리합니다.

Security & Reliability

제품화 완성도

비용 발생 작업을 실제 서비스 기준으로 보호합니다

생성·편집은 외부 AI 비용이 드는 작업이므로 인증, 과금, 입력 검증, 환불 정책이 같은 경로에서 처리됩니다.

Credit Ledger

요청 전 차감, 전체 실패 시 환불. 가격표는 Supabase의 pricing_config로 운영자가 조정합니다.

API Guard

Origin 검증, 인증 확인, per-user rate limit, 이미지 매직 바이트 검증, SSRF 방어를 적용했습니다.

Test Coverage

보안 가드, 마커 검증, 모델 요청 payload, 이미지 출력 정규화를 Vitest로 검증합니다.
