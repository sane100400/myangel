# MyAngel 3분 데모 발표 큐시트

목표: 심사위원에게 "AI 이미지 생성 래퍼"가 아니라, 한국어 프롬프트 강화 + 멀티모델 + 마커 정밀 편집 + 운영 가능한 크레딧 시스템이 한 흐름으로 연결된 제품임을 3분 안에 보여준다.

발표 자료: [DEMO_3MIN_SLIDES.html](./DEMO_3MIN_SLIDES.html)

## 사전 준비

- 라이브: https://ku-myangel.site
- 로그인된 브라우저 1개 준비
- 계정 잔액 200+ 크레딧 확인
- `/generate`, `/edit`, `/boards`를 각각 탭으로 미리 열어둠
- 시연용 베이스 이미지 1장, 레퍼런스 이미지 1장 준비
- 백업: `docs/DEMO.md`의 장애 대응 멘트

## 0:00-0:20 오프닝

"MyAngel은 한국어 사용자에게 맞춘 AI 이미지 생성·편집 스튜디오입니다. 핵심은 네 가지입니다. 프롬프트를 더 시각적으로 다듬고, OpenAI와 Google 모델을 한 화면에서 고르고, 원하는 영역만 마커로 편집하고, 결과 저장과 크레딧 운영까지 한 흐름으로 처리합니다."

보여줄 화면: 슬라이드 1

## 0:20-0:45 문제 정의

"일반적인 이미지 생성 도구는 프롬프트가 애매하면 결과가 흔들리고, 부분 편집에서는 모델이 '어느 의자'인지 매번 다르게 해석합니다. 데모에서는 이 두 문제, 즉 표현의 애매함과 편집 위치의 불확실성을 줄이는 흐름을 보여드리겠습니다."

보여줄 화면: 슬라이드 2

## 0:45-1:15 생성 플로우

"먼저 `/generate`입니다. '예쁜 카페의 감성적인 분위기'처럼 약한 표현을 넣고 프롬프트 강화를 누르면, AI가 바꿀 수 있는 표현을 잡아줍니다. 추천 표현을 선택하고 다듬기를 누르면 더 시각적인 문장으로 정리됩니다. 그 다음 GPT Image 2 또는 Nano Banana Pro, 화질과 장수를 골라 생성합니다."

보여줄 화면: `/generate` 또는 슬라이드 3

## 1:15-2:05 마커 편집 플로우

"결과에서 편집을 누르면 `/edit`로 넘어갑니다. 여기서 핵심은 말로만 편집하는 게 아니라, 이미지 위에 교체·추가·삭제 마커를 직접 찍는 점입니다. 레퍼런스 이미지를 올리면 현재 마커에 바로 연결되고, 편집 실행 시 모델이 위치와 의도를 함께 받습니다."

"Nano Banana Pro 경로는 원본, 마커 가이드, 그레이스케일 마스크를 같이 보내고, GPT Image 2 경로는 알파 마스크를 사용합니다. 사용자는 그냥 원을 찍지만, 내부에서는 모델별로 맞는 입력 계약을 분리합니다."

보여줄 화면: `/edit` 또는 슬라이드 4

## 2:05-2:35 운영/신뢰성

"서비스 운영 측면도 넣었습니다. 생성·편집은 크레딧을 먼저 차감하고, 전체 실패 시 자동 환불합니다. 저장 이미지는 `/boards`에 자동 보관되고, 공개하고 싶은 이미지는 Discover로 공유할 수 있습니다. 보안은 Supabase RLS, Origin 검증, 이미지 매직 바이트 검증, SSRF 방어를 적용했습니다."

보여줄 화면: `/boards` 또는 슬라이드 5

## 2:35-3:00 클로징

"정리하면 MyAngel은 멀티모델 생성, 한국어 프롬프트 강화, 마커 기반 정밀 편집, 저장·공유·크레딧 운영을 하나의 데모 가능한 제품으로 연결했습니다. 단순 API 호출보다 중요한 건 사용자가 원하는 이미지를 더 적은 시행착오로 만들게 해주는 워크플로우입니다."

보여줄 화면: 슬라이드 6

## 핵심 한 줄

"MyAngel은 프롬프트의 애매함과 부분 편집의 위치 불확실성을 줄여주는 한국어 AI 이미지 스튜디오입니다."
